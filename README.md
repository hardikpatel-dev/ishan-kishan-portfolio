# Ishan Kishan — Portfolio

A cinematic single-page portfolio for cricketer **Ishan Kishan**, built with Next.js, Tailwind, and Framer Motion. Features a layered wordmark hero with a paint-reveal interaction and scroll-driven section animations.

## Tech stack

- **Next.js 15** (App Router) + React 19 + TypeScript
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Lenis** for smooth scroll

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev        # dev server
npm run build      # production build
npm start          # serve production build
npm run lint       # lint
```

## Structure

```
components/
  sections/    # Hero, About, Journey, Stats, Teams, Moments, Gallery, News
  ui/          # PaintReveal and other reusable components
public/hero/   # Hero images
```
